# 🔮 Sentinel Visualizer

&gt; See the Unseen in ZK Circuits

Hardware-accelerated visual debugger for Zero-Knowledge circuits. 
Powered by [TOTAL Protocol](https://github.com/karamik/totalprotocol).

![Demo](docs/demo.gif) *(coming soon)*

## ✨ Features

- 🕸️ **3D Circuit Graph** — Interactive visualization of constraint dependencies
- 🔥 **Constraint Heatmap** — Identify expensive operations instantly  
- ⏱️ **Hardware Preview** — Simulate FPGA execution before deployment
- 🐛 **Step Debugger** — Trace witness generation step-by-step

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/karamik/sentinel-visualizer.git
cd sentinel-visualizer

# Install
pnpm install

# Run web app
pnpm dev
